# 🧠 Memory Mate

**Memory Mate** is your memory’s best friend – a mobile app designed to support individuals with memory loss, cognitive decline, or early-stage dementia. The app offers a user-friendly and accessible interface tailored to older adults and caregivers, aiming to improve quality of life through gentle cognitive stimulation, reminders, and essential services.

Memory Mate addresses the growing global challenge of dementia by providing engaging tools for users and streamlining caregiving responsibilities. It supports both users and their families with features like real-time face recognition, reminders, emergency services, and cognitive activities — helping promote independence and connection.
